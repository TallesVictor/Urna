# Urna
Esse projeto, tem o intuito de simular uma urna, onde os votos serão contados, 
caso haja eleitor para o número que o usuário digitar.
Exemplo: Se a pessoa digitar 00, vai aparecer o nome de quem está relacionado ao número 00.
O voto sera contado no banco de dados, e depois irá mostrar uma lista, em orde descrecente do número de votos.

Linguagem de Programação: Java.
Recursos utilizados: MVC e POO. 
Banco: MYSQL.
